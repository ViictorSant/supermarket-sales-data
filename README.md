## Sobre o projeto
Esse projeto se trata de uma Análise Exploratória (EDA) de um dataset de vendas de supermercado.
O objetivo é identificar padrões de consumo, comportamento de clientes, produtos mais vendidos e etc.
essa é a primeira versão do projeto e por isso utilizarei apenas o pandas por enquanto, pode ser que eu utilize outras bibliotecas para visualização no futuro.
